# VidScreener -- Business Model

## Overview

VidScreener monetizes by aligning pricing with the core value it
provides: - Faster evaluation - Reduced human effort - Improved
consistency and decision quality

The model combines **usage-based pricing (per video)** with **SaaS
tiers**, **marketplace revenue**, and **enterprise analytics**.

------------------------------------------------------------------------

## Core Principle

> Charge based on evaluation volume and value delivered, not just
> access.

------------------------------------------------------------------------

## 1. Per-Video Pricing (Primary Revenue Driver)

### AI Evaluation Pricing

-   ₹10--20 per video (AI-only evaluation)
-   ₹30--50 per video (AI + evaluator workflow tools)

### Why This Works

-   Directly tied to customer value
-   Scales naturally with usage
-   Easy to justify (saves evaluator time)

------------------------------------------------------------------------

## 2. Tiered SaaS Model

### 🟢 Free Tier (Adoption Layer)

-   Custom rubrics ✅ (always free)
-   Custom submission forms ✅ (always free)
-   Manual evaluation ✅ (always free)
-   Limited usage (e.g., 50 videos)
-   No or minimal AI features

**Goal:** Onboard organizations into the workflow

------------------------------------------------------------------------

### 🔵 Pro Tier (AI Layer)

-   AI-powered evaluation
-   Monthly quota (e.g., up to 500 videos)
-   Basic analytics
-   Standard support

**Target:** Colleges, small programs

------------------------------------------------------------------------

### 🟣 Enterprise Tier (High-Value Layer)

-   High-volume or unlimited video processing
-   Advanced analytics:
    -   AI vs Human calibration
    -   Rubric-level insights
    -   Evaluator performance
-   API access
-   Priority support
-   Optional private deployment

**Target:** Universities, large hiring pipelines, fellowships

------------------------------------------------------------------------

## 3. Evaluator Marketplace Revenue

### Model

-   Admins can hire external (open) evaluators
-   Platform takes a **10--20% commission**

### Example

-   Evaluator paid ₹100/video
-   Platform earns ₹10--20/video

### Value

-   Enables scaling evaluation capacity
-   Creates a two-sided marketplace
-   Adds recurring revenue stream

------------------------------------------------------------------------

## 4. Premium Analytics (Upsell)

Charge for advanced insights: - AI vs human agreement trends - Bias
detection (future) - Rubric reliability metrics - Evaluator consistency
tracking

### Why This Matters

-   High-margin
-   Valuable for decision-making
-   Hard to replicate

------------------------------------------------------------------------

## 5. API & Integration Pricing

For enterprise customers: - ATS integrations - Bulk ingestion
pipelines - Automated workflows

### Pricing Options

-   Per API call
-   Enterprise contracts

------------------------------------------------------------------------

## 6. Example Revenue Scenario

### Per Organization

-   1000 submissions
-   ₹20/video → ₹20,000

Add: - Marketplace cut - Analytics upgrades

→ ₹50,000 -- ₹1,00,000 per evaluation cycle

------------------------------------------------------------------------

## 7. Target Customers

### Primary

-   Universities (e.g., YCL-type programs)
-   Fellowships and accelerators

### Secondary

-   Hiring pipelines (later stage)

------------------------------------------------------------------------

## 8. What Makes This Model Strong

-   Usage-based → fair and scalable
-   Hybrid SaaS + marketplace → multiple revenue streams
-   High-value enterprise upsells
-   Strong alignment with customer pain

------------------------------------------------------------------------

## 9. What to Avoid

-   Pure subscription pricing (doesn't match usage variability)
-   Underpricing AI evaluation
-   Ignoring enterprise segment early

------------------------------------------------------------------------

## 10. Strategy

Start with: - Per-video pricing - Pilot with 1--2 institutions

Then expand: - Add marketplace - Add analytics upsells - Scale to
multiple organizations

------------------------------------------------------------------------

## Summary

VidScreener's business model is built around: - Per-video monetization -
Tiered SaaS access - Evaluator marketplace commissions - Premium
analytics and integrations

This creates a scalable, multi-layered revenue system aligned with
real-world evaluation workflows.
